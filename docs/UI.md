大まかに

* タイトル
* 画像エリア
* UIエリア

に縦に並ぶスマホ前提レイアウト

UIエリアには
* 上段
* 下段
がある。

上段には
* ペン
* スタンプ
* ホログラム
がある。

下段は上段のモードに応じて切り替わる
* ペンのとき
    * 線の太さ
    * ペンの種類
* スタンプ
    * スタンプの大きさ
    * スタンプの種類
* ホログラム
    * なし

UIエリアの左端には上段下段ぶち抜きで画像読み込みボタンがある
UIエリアの右端には上段下段ぶち抜きで画像作成ボタンがある

画像作成ボタンを押すとポップアップで画像のプレビューが表示される
