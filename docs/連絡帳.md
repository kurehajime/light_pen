# 連絡帳

- 画像アップロード/描画/UltraHDR生成のUI実装、`libultrahdr-wasm`のローカル依存手順を追加。
- libultrahdr-wasmのMeson/Emscriptenビルド手順とMakefileを整備し、`libultrahdr-esm`のみビルドするよう調整。
- `appendGainMap`を使ったUltraHDR生成に変更し、ImageData→JPEG変換を追加。
- GitHub Pages向けにUltraHDRモジュールの読み込みパスを`BASE_URL`対応に修正。
- UltraHDR生成後のプレビュー表示を追加。
- ペンの最大太さを150pxに調整。
- ダウンロードボタンを撤去し、プレビュー表示のみ残すよう整理。
- プレビューURLの破棄処理での参照名ズレを修正。
- ホログラム画像を線レイヤーへ反映するボタンを追加し、全画面タイル敷きで模様を適用。UltraHDR生成は1ボタンに統一。
- ホログラム明部の強調度を上げ、`gainMapMax`/`hdrCapacityMax` を64に調整。
- `ペン` `判子` `効果` のモードボタンを連結見えに変更し、左右端だけ角丸になるよう境界を調整。
- ボタン全体の立体感を強め、各ボタンの枠線と影色を背景色の系統に合わせて調整。モード選択中は凹み表現に変更。
- 下段の排他UIも上段モードと同系統の押し込み表現に統一し、選択色をピンク系へ調整。`読込` `戻す` `作成` も同系色に揃えたうえで角丸形状に整理。
- UIボックス類にくっきりした内側の凹み影を追加し、画像選択エリアのアクセント色も緑系からピンク系へ統一。
- 通常ボタンに薄いグラデーションを追加しつつ、選択状態のデザインは維持。ホバー時は浮き上がりではなく軽く凹む挙動に変更。
- 画像読込前の画像エリアをビフォーアフター紹介用コンポーネントへ分離し、サンプル画像と読込ボタンをスマホ/PC両対応のレイアウトで表示。読込後にサンプルが消えるよう表示切替も修正。
- ビフォーアフター導線の矢印色を濃いピンクへ変更し、サンプル画像が大きく見えるよう余白を調整。上下余白は維持しつつ左右を狭めた。
- UI全体で使う色と影・グラデーション定義を`src/index.css`の`:root`へ集約し、`App.css`側は参照ベースに整理。矢印色もCSS管理へ移行。
- 未使用の色変数を削除し、同じ色を指す変数も統合してカラートークン構成を整理。
- 色数を絞るため、ピンク系トークンの一部を`--button-main`起点の`color-mix()`算出へ変更。あわせて下段パネルに最低高さを持たせ、モード切替時のスクロール揺れを抑制。
