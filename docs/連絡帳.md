# 連絡帳

- 画像アップロード/描画/UltraHDR生成のUI実装、`libultrahdr-wasm`のローカル依存手順を追加。
- libultrahdr-wasmのMeson/Emscriptenビルド手順とMakefileを整備し、`libultrahdr-esm`のみビルドするよう調整。
- `appendGainMap`を使ったUltraHDR生成に変更し、ImageData→JPEG変換を追加。
- GitHub Pages向けにUltraHDRモジュールの読み込みパスを`BASE_URL`対応に修正。
- UltraHDR生成後のプレビュー表示を追加。
- ペンの最大太さを150pxに調整。
- ダウンロードボタンを撤去し、プレビュー表示のみ残すよう整理。
- プレビューURLの破棄処理での参照名ズレを修正。
- ホログラム画像を線レイヤーへ反映するボタンを追加し、全画面タイル敷きで模様を適用。UltraHDR生成は1ボタンに統一。
- ホログラム明部の強調度を上げ、`gainMapMax`/`hdrCapacityMax` を64に調整。
- `ペン` `判子` `効果` のモードボタンを連結見えに変更し、左右端だけ角丸になるよう境界を調整。
- ボタン全体の立体感を強め、各ボタンの枠線と影色を背景色の系統に合わせて調整。モード選択中は凹み表現に変更。
- 下段の排他UIも上段モードと同系統の押し込み表現に統一し、選択色をピンク系へ調整。`読込` `戻す` `作成` も同系色に揃えたうえで角丸形状に整理。
- UIボックス類にくっきりした内側の凹み影を追加し、画像選択エリアのアクセント色も緑系からピンク系へ統一。
- 通常ボタンに薄いグラデーションを追加しつつ、選択状態のデザインは維持。ホバー時は浮き上がりではなく軽く凹む挙動に変更。
- 画像読込前の画像エリアをビフォーアフター紹介用コンポーネントへ分離し、サンプル画像と読込ボタンをスマホ/PC両対応のレイアウトで表示。読込後にサンプルが消えるよう表示切替も修正。
- ビフォーアフター導線の矢印色を濃いピンクへ変更し、サンプル画像が大きく見えるよう余白を調整。上下余白は維持しつつ左右を狭めた。
- UI全体で使う色と影・グラデーション定義を`src/index.css`の`:root`へ集約し、`App.css`側は参照ベースに整理。矢印色もCSS管理へ移行。
- 未使用の色変数を削除し、同じ色を指す変数も統合してカラートークン構成を整理。
- 色数を絞るため、ピンク系トークンの一部を`--button-main`起点の`color-mix()`算出へ変更。あわせて下段パネルに最低高さを持たせ、モード切替時のスクロール揺れを抑制。
- `logo.jpeg`はタイトル画像ではなく小さいアイコンとして使う形に修正し、タイトル文字を復帰。あわせてタップ時の誤テキスト選択を画像エリア含む非編集UIで抑止し、ハート/スターのペン描画密度も少し下げて粒数上限を3に調整。
- `dynamic-range: high` の HDR 対応判定を追加し、未対応環境では画像読込前エリアに「キラキラはHDR対応環境でのみ表示されます」の注意文を文字だけで表示するようにした。
- `react-i18next` で日本語/英語表示に対応し、`lang=en` クエリ時は英語を強制。あわせて `読込` `作成` `画像をえらぶ` の強調ピンクと白文字/白アイコンの影を調整し、通常時の文字色と主要SVGアイコン色もタイトルと同じ濃い紫へ統一。
- 強調ボタンの彩度を少し上げ、`読込` `作成` `画像をえらぶ` の白文字は細い影ではなく太めの紫縁取り文字に変更して視認性を調整。
- 強調ボタンの表現を外側発光から内側の艶がゆっくり流れるアニメーションへ変更し、下段の`読込`ボタンは通常ボタン扱いに戻して`画像をえらぶ`のみを読込導線の強調対象に整理。

## 追記漏れのUI調整

- 上段モードボタンと下段の選択UIは、選択時に背景色と文字色を反転させるルールへ統一し、押し込み量や影の見え方も揃えた。
- ボタンのテカり表現は撤去し、通常ボタンの罫線・段差・アイコン色、レンジのアクセント色、各UIボックスの基準色をテーマ変数ベースで整理した。
- 単色SVGアイコンは`src/icons`配下のReactコンポーネントへ移行し、CSSの`color`で制御できるように変更。未使用の`public`内SVGも削除した。
- 完成ボタンと読込ボタンのアイコンは指定SVGベースの`CheckIcon`/`ImageIcon`へ差し替え、作成ボタンの文言も`完成`/`完成中`へ変更した。
- カラーパレットは複数回調整し、最終的に背景`#EDEC8F`、文字とボーダー`#f44182`、通常ボタン`#ECD6E2`、強調ボタン`#f35353`を基準に統一した。
- 背景の星/ハート柄は撤去して無地にし、タイトル文字の白縁を外した。タイトル横ロゴの角丸も撤去した。
- UIボックスや画像エリアは、ぼかし影ベースの浮き表現から太い罫線ベースへ変更し、画像エリアとボタンボックスの間隔をなくして一体化した。接合部の二重線は上段ボックスの上罫線を外して解消した。
- 画像エリアとボタンボックスの角丸は見直し、全体から一度角丸を撤去した後、ボタン本体とボタンボックスだけ角丸を復帰。ボタンボックスは上側のみ角丸なしに整理した。
- 下段のラジオ系ボタンには左端の選択マークを追加し、未選択時は丸、選択時は通常ボタン配色の丸と中点で状態が分かるように変更。エフェクトボタンはラジオ表現から除外した。
- 強調ボタンは、テカりアニメーションを試したのち撤去し、最終的に浅く定期的に押し込まれるアニメーションと、ホバー時に押し込まれた状態へ移る挙動へ変更した。
- 完成ボタン押下時は直接生成せず、下段パネルを生成メニュー表示へ切り替えて `画像を生成` と `X向け画像を生成` を出す形に変更。通常生成のみ既存処理へ接続し、X向けは未実装のため無効化した。
- `hdr_sandbox/light_pen` のロジックを元に、`X向け画像を生成` で ICC 埋め込み付き HDR PNG を1枚だけ生成する処理を移植。`pako` と `@imagemagick/magick-wasm` を追加し、生成時のみ動的ロードする構成にした。
- 生成メニューの2ボタンに `ImageIcon` を追加し、完成ボタンはトグルではなく独立モードとして扱うよう変更。選択中は他モードと同じ反転配色になるよう文字色・アイコン色のCSSも調整した。
